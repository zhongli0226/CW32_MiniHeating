# 项目介绍
这是个利用PD协议实现的恒温加热台，主控采用CW32F030，这个是国产32，有幸免费样品申请，申请到了所有用其作为主控制作一个项目。项目主要借鉴来源于[[PD协议 | 高颜值\]mini加热台 - 嘉立创EDA开源硬件平台 (oshwhub.com)](https://oshwhub.com/littleoandlittleq/bian-xie-jia-re-tai)

# 软件架构
开发环境 VScode+gcc编译开发环境，
